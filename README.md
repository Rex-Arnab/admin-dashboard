# NextJs 14 Admin Dashboard - NextAdmin

![readme_image](public/readme_image.png)

This is made using the following stack:
- Framework - Next.js 14
- Language - TypeScript
- Auth - NextAuth.js
- Database - Vercel Postgres
- Deployment - Vercel
- Styling - Tailwind CSS
- Components - Tremor
- Analytics - Vercel Analytics
- Linting - ESLint
- Formatting - Prettier
